## Overview

Voltivity (λ, units: V·μm) is a material constant that quantifies the threshold for resonant coupling between applied electric fields and phonon modes in crystalline solids. For any material, the product of the applied electric field and the characteristic microstructural interaction length at flash onset remains invariant:

```
λ = E × r = constant for each material
```

This repository contains:

- **Complete 68-point validation dataset** spanning 36 materials across 17 crystal structure families
- **Statistical analysis scripts** reproducing all key results from the paper
- **Figure generation code** for the universal scaling collapse and prediction accuracy plots

## Dataset

`data/voltivity_dataset.csv` — 68 experimental measurements compiled from 34 peer-reviewed publications.

| Column | Description | Units |
|--------|-------------|-------|
| `Material` | Material composition | — |
| `Family` | Crystal structure family | — |
| `E(V/cm)` | Applied electric field at flash onset | V/cm |
| `T_onset(K)` | Measured onset temperature | K |
| `r_fitted(um)` | Energy localization length | μm |
| `lambda(V·um)` | Voltivity: E × r | V·μm |
| `k_soft` | Phonon softening factor | — |
| `Ea(eV)` | Activation energy for conduction | eV |
| `sigma_0(S/m)` | Pre-exponential conductivity | S/m |
| `r_calc(um)` | Predicted localization length | μm |
| `T_pred(K)` | Predicted onset temperature | K |
| `T_error%` | Prediction error | % |
| `DOI` | Source publication DOI | — |

### Key statistics

- **36 materials**, 17 crystal structure families, 34 source publications
- **λ range:** 1,000–27,630 V·μm (metals → covalent carbides)
- **Within-material CV:** 1.9% mean (for materials with ≥2 measurements)
- **Prediction accuracy:** 2.6% mean |error| on onset temperature

## Quick Start

```bash
# Install dependencies
pip install numpy pandas matplotlib

# Run validation analysis
python scripts/validate_lambda.py
```

This reproduces the key statistical results and generates three figures in `figures/`.
